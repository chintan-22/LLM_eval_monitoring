🔴 CRITICAL: Must Change (3 Steps - 10 Minutes)
──────────────────────────────────────────────────────────────────────────────

1️⃣ CREATE .env FILE
   Location: Project root
   Action:
     $ cp .env.example .env
     $ nano .env
   
   Change these values:
     MODEL_NAME=your-model                    (e.g., gpt-3.5-turbo)
     MODEL_PROVIDER=your-provider             (e.g., openai)
     FAITHFULNESS_THRESHOLD=0.75              (your quality standard)
     ANSWER_RELEVANCY_THRESHOLD=0.70          (your quality standard)
     CONTEXT_PRECISION_THRESHOLD=0.65         (your quality standard)
     SEMANTIC_SIMILARITY_THRESHOLD=0.60       (your quality standard)


2️⃣ REPLACE GOLDEN DATASET
   Location: data/golden_dataset.json
   Current: 5 investment Q&A pairs
   Action: Replace with YOUR Q&A pairs
   
   Keep this structure:
     {
       "question": "YOUR QUESTION",
       "retrieved_context": "YOUR CONTEXT",
       "ground_truth": "EXPECTED ANSWER",
       "model_answer": "ACTUAL ANSWER",
       "metadata": {"category": "...", "difficulty": "..."}
     }
   
   Need at least 5 examples


3️⃣ TEST IT
   $ python scripts/run_eval.py
   
   Should see:
     ✓ Configuration loaded

     ✓ Evaluation complete
     ✓ Metrics computed

⚡ RAPID REFERENCE
──────────────────────────────────────────────────────────────────────────────

What is this project?
  → LLM Evaluation & Monitoring Pipeline
  → Evaluates LLM response quality
  → Detects performance degradation
  → Sends alerts when quality drops

What do I need to change?
  → .env file (configuration)
  → data/golden_dataset.json (your Q&A data)
  → Everything else is optional

How long does setup take?
  → Minimum: 5 minutes
  → Recommended: 30 minutes
  → Full customization: 2-3 hours

What are the quality metrics?
  → Faithfulness (how accurate to context)
  → Answer Relevancy (how relevant to question)
  → Context Precision (how relevant is context)
  → Semantic Similarity (how close to ground truth)

How do I know if it's working?
  → Run: python scripts/run_eval.py
  → Should see: 4 metrics × N samples evaluated
  → Check logs: tail logs/evaluation.log
